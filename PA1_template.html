Imputing missing values

  1. Calculate and report the total number of missing values in the dataset. +Total rows with NAs:
print(sum(is.na(data$steps)))
## [1] 2304
  1. Fill in missing values using average steps per day across the dataset.
list <-which(is.na(data$steps))
listlength <-length(list)
stepsonaverage <-with(data, tapply(steps, date, mean, na.rm=TRUE))
replace <-mean(stepsonaverage,na.rm = TRUE)
for (i in 1:listlength) {
        data[list[i],1]<-replace
}
